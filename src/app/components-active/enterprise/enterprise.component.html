Welcome to Benny's Enterprise Component Library!

The following content is a heavily augmented version of the internal component library that I have founded and maintained in DBS Bank.
Table
Data tables are integral to most enterprise products. Throughout my career, I have designed and built reusable table components which feature many dynamic and bespoke functions. These components have been published to company-wide component libraries.

The demo below is a simple sortable CRUD table that can perform inline edit and delete operations. It was built entirely with Angular Material and PrimeNG libraries.
Buttons
While building applications for enterprise use, buttons come in different colours and sizes to convey different functions. This component condenses all the styling and functional requirements to a one-line, plug-and-play solution.
